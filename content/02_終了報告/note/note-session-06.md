# ゲームクリエイター教室 体験会を終えて（第6回）

ブロマスとして開催しているゲームクリエイター教室の体験会、第4回・第5回を実施しました。

今回は、Robloxを3回、AIを2回開催。特にRobloxは人気コンテンツということもあり、多くの親御さんにご参加いただきました。

体験内容は、お子さん一人ひとりの「作りたい」に合わせて調整しています。

- プログラミングに挑戦したいお子さんには「アイテムのカウントギミック」
- 世界観を表現したいお子さんには「アスレチック制作」
- 鉄道好きなお子さんには「鉄道をテーマにした制作」

それぞれの興味に寄り添いながら、個別最適な体験を提供できたと感じています。

また今回は体験会のコンテンツを増やしたことで、リピート参加してくれたご家族もいらっしゃいました。さらに、遠方からお越しいただいた方もおり、全体として非常に盛況な回となりました。

## AIワークショップについて

AIの回では、お子さんだけでなく親御さんも一緒に体験される場面が多く見られました。

「自分の思い通りにカスタマイズしていく体験」は、世代を問わず楽しんでいただけた印象です。

一方で、使用回数の制限により進行がバタつく場面もありました。今後はこの点を改善し、よりスムーズな体験設計を行っていきます。

## 今、考えていること

「ゲームクリエイターになりたい」という変わらない夢。そして、AIによって大きく変わりつつある制作環境。

この2つをどう組み合わせ、学びの場として提供していくか。それが、今の自分に課されているテーマだと感じています。

## 今後について

現在、一般的なプログラミング教室や塾とは少し異なる取り組みも進めています。詳細はまだお伝えできませんが、より特徴ある教室へと進化していく予定です。

今後の展開も、ぜひ楽しみにしていただければ嬉しいです。
