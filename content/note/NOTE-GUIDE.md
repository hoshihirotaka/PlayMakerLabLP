# ブロマスゲームラボ note 制作ガイド

noteはファネル上「信頼構築・取り組みの深さ」を担うチャネル。  
インスタが雰囲気訴求なのに対し、noteは「なぜやっているか」「現場で何が起きているか」を伝える場。

---

## ファイル命名規則

```
note-session-XX.md     # 活動報告（回数）
note-YYYY-MM-DD.md     # 回数に紐づかない単発記事
```

---

## 記事の種類と使い分け

| 種類 | 頻度 | 内容 |
|------|------|------|
| 活動報告 | 毎回開催後 | 当日の様子・反省・気づき |
| 考察・コラム | 不定期 | ゲームデザイン思考、AI活用、教育観など |
| お知らせ | イベント前後 | 開催情報・募集（インスタの補足として） |

---

## 文章トーンのルール

- **塾のチラシ調にしない**。開発現場のライブ感、試行錯誤の正直さを大事にする
- 反省点も隠さず書く。「うまくいかなかった→こう改善する」の流れが信頼につながる
- 子どものエピソードは具体的に。「〇〇なお子さんが〇〇できた」が読者の心に刺さる
- 親御さん視点も意識する（35〜44歳、「消費より創造」を求める層）
- 締めは次回への予告または展望で終わる

## 書かないこと

- 個人が特定できる情報（名前・学校名など）
- 断定的な成果（「必ず〇〇になれます」系）
- 他の教室・サービスの名指し批判

- [ ] 次回の日時・形式は最新情報か
- [ ] インスタ投稿文（`content/social/`）と日程表記が一致しているか
