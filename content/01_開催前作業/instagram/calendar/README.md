# Instagram 月次カレンダー告知

月の開催日程をまとめて案内する投稿文。イベントごとの募集投稿（`instagram-post-YYYY-MM-DD.md`）とは別。

## 命名

```
instagram-calendar-YYYY-MM.md
```

例: `instagram-calendar-2026-07.md`（7月分）

テンプレート: [instagram-calendar-template.md](instagram-calendar-template.md)

## 使い方

1. テンプレートをコピーして `instagram-calendar-YYYY-MM.md` を作る（または前月分を複製）
2. 日付・時間（`HH:MM－HH:MM`）・会場を埋める
3. 個別イベントの申込URLがまだない段階では「追ってインスタでご案内」で締めてよい
4. 投稿後もファイルは残す（翌月の参考になる）
