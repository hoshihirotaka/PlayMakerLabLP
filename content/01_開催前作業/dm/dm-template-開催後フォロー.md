# テンプレート：開催後フォロー①（参加者向け）

- **配信面**: その回の参加者宛（Doorkeeperの一斉メッセージ／日付指定で予約送信）
- **タイミング**: 参加翌日〜2日後
- **狙い（本命）**: 「体験＝入口」だと明言し、教材の幅を見せて「1回で終わり」誤解を解く。
- **タイトル（件名）**: `【ブロマス】ご参加ありがとうございました（M/D エリア）`  
  例: 【ブロマス】ご参加ありがとうございました（7/18 浦和）

## ナレッジ（2026-07時点）

- 問題: 体験参加者が「1回目以降にも教材がある」と知らない → 成約B低下の一因。
- **本命リンクはカリキュラム一覧1本**（作品Instagramや継続プランURLは①に入れない）。
- 次回予告・カレンダー・作品紹介は**全体メール**側。参加者向けの7日後ステップは、現状のDoorkeeper運用ではやらない。
- 教材の幅はジャンル列挙より、親が選べる軸で書く:
  1. 簡単なプログラミングを始める
  2. プログラミングなしの演出・モノづくり
  3. AIでアイデアを形にする
- CTAは「リンクを見て、どれが良さそうかお子様と会話してみてください」。締めは「お問い合わせください」（メール返信に限定しない）。

**差し込み変数**
- `{{日付エリア}}` … 例: 7/18 浦和
- `{{会場一言}}` … 例: 浦和での（なければ空欄で「ワークショップに」）
- `{{当日の教材}}` … 例: 探し物・レース／コース（複数コースや固有名を書かないとき）
- `{{カリキュラムリンク}}` … utmは `utm_medium=followup`  
  `https://hoshihirotaka.github.io/PlayMakerLabLP/?utm_source=doorkeeper&utm_medium=followup#curriculum`

- タイトル: 【ブロマス】ご参加ありがとうございました（{{日付エリア}}）
