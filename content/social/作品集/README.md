# 作品集インスタ投稿ワークフロー

## フォルダ構成

```
content/social/
  お知らせ/   ← イベント告知・募集投稿
  作品集/     ← 参加者の制作物紹介投稿（このフォルダ）
```

## Canva → Claude → 投稿文のワークフロー

### 1. Canvaからエクスポート

**PDF（推奨）**を使う。

- CanvaでPDFダウンロード → Claudeに渡す
- テキスト＋ビジュアル（スライドデザイン）両方読み取れる
- 複数スライドも1ファイルでOK

| 形式 | テキスト | ビジュアル | 備考 |
|------|----------|------------|------|
| PDF  | ✅ | ✅ | **推奨** |
| 画像（PNG/JPG） | ✅ | ✅ | 1枚ならOK・複数は手間 |
| PPTX | ✅ | ❌ | 画像・デザインが取れない |

### 2. Claudeに渡す

PDFをアップロードして「インスタの文言作って」と伝えるだけ。

### 3. 投稿文ファイルの命名規則

```
instagram-works-YYYY-MM-DD.md  ← 制作日（作品情報のスライドに記載）
```

### 4. 投稿文の構成（テンプレート）

```
🎮今回の作品は...

[作品の特徴・仕掛けを2〜3行]

[作者紹介（「ワークショップに来てくれた小学生」など）]

📍ブロマスゲームラボ（さいたま・南与野）
体験会の詳細はプロフィールリンクから↑

#ブロマスゲームラボ #[ツール名] #子どもの作品 #ゲーム制作 #小学生 #親子参加 #さいたま #南与野 #ゲームクリエイター
```

### 5. ポイント

- ひったさんのコメント（作品評）を文章の核にする
- ゲームの「仕掛け」や「こだわり」を具体的に書く
- タイトルがなくてもOK（それ自体がネタになる）
- 「連載感」を出す冒頭は初回のみ使用済み。2回目以降は「今回の作品は...」から
