## LPから教材リポジトリへの依頼・フィードバック

### 表示側の仕様（守ってほしい点・現状は満たされている）
- `summary` は40字以内目安。現状の16件は収まっており問題なし。
- `detail` は120字以内目安。現状OK。
- `icon` は絵文字1つ推奨。現状は準備中含め全件付与されており良好。
- `sessions` は数値・文字列どちらでも表示できる（例「9種・全21回」「未定」も可）。

### 🔴 status readiness補正（2026-07-14・LP側で先行修正）
前回の `curriculum.json` は「リポジトリにファイルがある＝published」と**過剰にpublished判定**されていた（教材側 `meta.note` でもアバターを「READMEでは未整備だが整備済みと判断」と申告＝実態とズレていた）。ひったさんに実際に出せるかを確認し、以下を **coming_soon に補正**（LP側で修正済み）：
- パーティゲーム / アバター変身 / テレイン（地形づくり）: まだ提供できる状態でない
- AI画像素材づくり / AI音楽づくり: **利用ツールに年齢制限**があり、子ども向けに使うには一手間（回避策）が必要
- → **教材側への依頼**: status の既定を `coming_soon` にし、**実際に提供できると確認できたものだけ `published`** にすること（次回同期でpublishedに戻さない）。上記5件は教材リポジトリ側の実態とも揃えてほしい。

出せる（published）と確定した10件: アスレチック / かくれんぼ / アクション / レーシング / UEFN体験 / オビー / たからさがし / メカニクス体験 / エフェクト / AIゲームづくり。
